StayMsg 릴리즈 노트
========================================

v1.0 — 기초 기능
----------------------------------------
[ 문자 템플릿 ]
- 예약 단계별 문자 템플릿 6단계
  (문의 / 예약전 / 예약확정 / 입실전 / 입실후 / 퇴실)
- 단계별 다중 템플릿 등록 및 전환
- {변수} 자동 치환 · 원터치 복사
- 자동 요금 계산 (주중/주말/성수기/연박할인/추가인원 반영)
- PWA (홈 화면 아이콘 설치 지원)

[ 공통 설정 ]
- 숙소명 · 주소 · 계좌번호 · 와이파이 SSID/비밀번호 설정
- 추가 인원 요금 설정 (기준 초과 1인당 요금)
- 강아지 동반 허용 여부 · 마릿수 제한 · 추가 요금 설정
- 바베큐 허용 여부 · 추가 요금 설정
- 특별 기간 설정 (날짜 공통 적용 · 성수기/비수기 등 기간명 지정)

[ 객실 개별 설정 ]
- 다중 객실 등록 및 탭 전환
- 객실명 설정
- 기준 인원 · 최대 인원 설정
- 주중 · 주말 · 기본 요금 설정
- 연박 할인 설정 (N박 이상 시 할인율)
- 특별 기간별 요금 (객실마다 별도 설정)


v1.1 — 템플릿 편의 기능
----------------------------------------
- 템플릿 입력창 예시: 내용 형태 플레이스홀더 (흐린 글씨)
- 템플릿 편집 시 {변수} 버튼으로 삽입 기능
- 템플릿 편집 시 이모지 선택 삽입 기능
- 헤더 서브타이틀 문구 수정 ("예약 단계별 문자 템플릿 · 자동 요금 계산")
- {성수기요금} 변수 데스크탑에서 오류 수정


v1.2 — 설정 구조 개편
----------------------------------------
- 기준/최대인원 공통설정 → 객실 개별설정으로 이동
- 저장 버튼 → 💾 플로피디스크 아이콘
- 복사 버튼 → 📋 클립보드 아이콘 (복사 후 자동 복원)
- 입실 날짜 선택 시 퇴실 날짜 자동 제한 (입실 이후만 선택 가능)


v1.3 — 손님 저장 기능
----------------------------------------
- 손님 이름 · 연락처 저장
  (예약 전 저장 시 이름/전화번호만, 예약 확정 후 저장 시 전체 숙박 정보 포함)
- 이름 2글자 · 전화번호 7자리 이상 입력 시 드롭다운 자동완성
- 동일 손님 다중 숙박 이력 저장 및 조회
- 미래 예약 건만 드롭다운 표시 (과거 숙박 제외)
- 여러 예약 있을 때 ▾ 서브 리스트 펼치기
- 예약확정 고객 이름 옆 연락처 표시


v1.4 — 모바일 디자인 리뉴얼
----------------------------------------
- 전체 디자인 모바일 퍼스트 개편 (화이트/클린, 코랄 포인트 #FF6B4A)
- 하단 탭 네비게이션 적용
- 카드형 레이아웃, Airbnb/당근 스타일


v1.5 — 모바일 버그 수정 & 앱 관리
----------------------------------------
- 입실·퇴실 날짜 한 줄 레이아웃 (→ 연결, 모바일 최적화)
- 모바일 날짜 입력창 달력 아이콘 표시 (PC는 숨김)
- 미취학아동 · 강아지 동반 항목 세로 배치 (텍스트 잘림 해결)
- 객실 탭 가로 스크롤 개선
- 🔄 최신버전 업데이트 버튼 헤더 추가 (캐시만 삭제, 데이터 유지)
- 🗑️ 데이터 초기화 버튼 설정 하단 추가 (확인 팝업 후 전체 초기화)
